== Description ==

THERING Plugin - The BEST WordpPress Circle Animated Popup Plugin

THERING Plugin is a very lightweight Plugin for Wordpress which allow users to add a customizable, circle animated pop-up.
It has a very user-friendly interface, which allow Wordpress users to easily modify settings needed for animated rings and box container, without coding.

The rings pictures, boxes and the enter button are separately customizable.

This plugin is basic-user oriented, no code knowledge is required.
Simply the best Wordpress circle animated popup plugin!


	★ Selcted pages where you want to display the circle animated popup, by PAGE TEMPLATE OPTIONS or SHORTCODE [thering]!
	★ Automatic centering in browser page.
	★ Fully and easy customizable aspect (background, color, picture etc...).
	★ Settable rings by rings custom post type content.
	★ Settable template part by plugin options.
	★ Custom text to display whatever you want with Ring Custom Post Type editor.
	★ Pretty fade-in and fade-out animation.(coming soon)

== Installation ==


LIKE IT? [LEAVE A 5 STAR REVIEW ;)]


1. Unzip `thering-plugin.zip` to the `/wp-content/plugins/` directory or select it by "Add new" in the 'Plugins' menu in WordPress
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to "Rings-> ajouter" in the admin menu and add yours ring content (title, content, link, picture...)
4. Go to "PlugRings" in the admin menu and set the plugin Template part options
5. Enjoy it



== Frequently Asked Questions ==


= Will the popup cause problems for SEO(Search Engine Optimization)? = 
No it is a simple div in your pages no meta tag no keywords, so it will not affect your search engine indexing.


= Does the plugin add some tables in my WordPress database? =
No. It's use wp_otpions only to store content.


= Can I create different plugins for different pages? =
No, you can’t.

= Is the popup responsive? =
Yes it is fully responsive.
